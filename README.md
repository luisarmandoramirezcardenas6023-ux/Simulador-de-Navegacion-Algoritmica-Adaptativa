# Simulador-de-Navegacion-Algoritmica-Adaptativa
Simulador de navegación urbana y logística de última milla en Python (NetworkX). Modela una red real de 259 nodos en Tijuana, aplicando IA adaptativa para optimizar rutas en tiempo real ante obstáculos dinámicos estocásticos. Incluye renderizado gráfico interactivo y un editor de topología para la gestión del grafo.
